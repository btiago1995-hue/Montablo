## Content Production Guidelines

### Blog post standards
- **Word count:** 800-1,500 words minimum
- **Structure:** H1 + H2s + H3s, short paragraphs, bullet lists
- **Images:** 2-3 per post, with descriptive alt text
- **Internal links:** Minimum 3 per post (to product pages, related posts)
- **CTA:** Every post ends with a signup CTA
- **Schema:** BlogPosting JSON-LD on every article
- **Meta:** Custom title tag (< 60 chars) and description (< 155 chars)

### Static page standards
- **Schema:** Appropriate structured data per page type
- **Breadcrumbs:** Visible + BreadcrumbList schema
- **Internal links:** Cross-link to related feature/blog pages
- **CTA:** Clear primary CTA above the fold

### Content quality checklist
- [ ] Addresses a specific search intent
- [ ] Provides unique value (not generic AI-generated fluff)
- [ ] Includes concrete examples or data
- [ ] Has clear, scannable structure
- [ ] Contains relevant internal links
- [ ] Has optimized meta tags
- [ ] Includes appropriate schema markup
- [ ] Mobile-friendly layout verified
